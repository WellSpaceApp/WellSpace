# Security Policy

## Supported Versions

WellSpace is a single continuously-deployed web app (no version numbers).
Only the current live version at wellspaceapp.github.io/WellSpace is supported.

## Reporting a Vulnerability

If you find a security vulnerability in WellSpace, please report it privately
rather than opening a public issue.

- Email: wellspacewebsite@gmail.com
- Please include steps to reproduce and, if possible, the impact.

I'll acknowledge reports within 48 hours and aim to patch confirmed
vulnerabilities within 7 days. Please give me time to fix an issue before
disclosing it publicly.
